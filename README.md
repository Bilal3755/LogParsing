Streamlit, py based webserver log parser that can be used to analyze apache/nginx logs.
Log size max that can be uploaded is up to 200MB
we can sort log based on timestamp, methods, keyword search etc
Provides IP and URL break down along with request count
#GUI code is not owned by me but is written with Gemini
Please review and use it with caution

reference dependencies need to run code
pip install streamlit pandas

To run code
 streamlit run logparser.py
